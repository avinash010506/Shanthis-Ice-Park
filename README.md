# Ice Park Delights

A premium, modern restaurant web application built with a focus on high-end design, smooth micro-animations, and responsive user experience. 

## 🌟 Features

- **Premium UI/UX:** Stunning "Gold-on-Dark" aesthetic featuring glassmorphic navigation bars, glowing call-to-actions, and elegant typography.
- **Dynamic Menu Catalog:** A fully categorized and structured digital menu covering everything from starters and pizzas to desserts and beverages.
- **Smooth Animations:** Integrated scroll-reveal effects, marquee banners, and fluid CSS transitions for a dynamic feel.
- **Fully Responsive:** Carefully crafted layouts that look perfect on desktop, tablet, and mobile devices.
- **Optimized Performance:** Lightning-fast build and hot module replacement (HMR) powered by Vite.

## 🛠 Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (with custom theme variables and utility classes)
- **Language:** TypeScript
- **UI Components:** Radix UI primitives & Lucide Icons
- **Animations:** Custom CSS keyframes & Framer Motion
- **Routing:** React Router v6

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (v18 or newer) installed on your machine.

### Installation
1. Clone the repository to your local machine.
2. Install the dependencies:
```bash
npm install
```

### Running Locally
To start the development server with hot-reload:
```bash
npm run dev
```
The application will be available at `http://localhost:8080/`.

### Building for Production
To create an optimized production build:
```bash
npm run build
```
The compiled output will be available in the `dist` directory.

## 🎨 Customization

The core theme colors and aesthetics are centrally managed in `src/index.css`. 
- Primary Theme: `hsl(43 74% 49%)` (Premium Gold)
- Background: `hsl(20 14% 8%)` (Rich Dark)

To modify the theme, simply update the CSS variables inside the `:root` and `.dark` blocks in `index.css`.
